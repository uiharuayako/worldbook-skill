# 世界观提取指南

从轻小说/游戏/小说等源材料中提取世界观设定，转化为世界书条目。

**核心原则：零度写作、数据库格式。去除所有小说式叙述、意象比喻和主观评价。**

---

## 内容格式

提取后的世界观条目采用 **XML 包裹 YAML** 格式：

```yaml
<world>
世界: xxx
核心规则:
  规则1: xxx
  规则2: xxx
区域:
  区域A: 定位描述
  区域B: 定位描述
</world>
```

---

## 提取维度

### 1. 世界观类型判断

先判断源材料的世界观属于哪类，这决定了提取量：

| 类型 | 特征 | 提取策略 |
|------|------|---------|
| A 真实背景 | 现代中国/日本/唐代等，AI默认已知 | 只提取AI不知道的虚构部分 |
| B 小世界 | 剑与魔法/赛博朋克/末日等，AI知道套路 | 提取与AI已知套路的差异 |
| C 大世界 | 完全原创的修仙/奇幻/星际等 | 完整提取所有设定 |

判断标准：**删掉这项设定后，AI是否还能正确表现这个世界？能→不提取。**

### 2. 核心设定与独特规则

提取构成世界基底的极简规则：
- 力量来源（灵气/魔法/科技/神赐）
- 等级/阶段划分
- 核心运行逻辑（如"修炼消耗寿命""魔法需要吟唱"）
- 自造词和专有概念

要求：
- 用最少字交代规则
- 区分"直接宣布的规则"和"通过角色体验间接展现的规则"
- 为自造词给出简短定义

### 3. 空间与势力架构

- 地理区域划分、各区域关键特征
- 势力集团（名称、定位、互相关系）
- 命名特色（冷峻/诗意/功能化等）

要求：
- 用数据库列表格式，不用游记散文
- 势力只写名称+一句话定位，不展开
- 区域描写如果原文是"风景抒情"，提取时去掉抒情，只留地理信息

### 4. 历史与时间线

- 影响当前世界格局的关键历史事件
- 时代背景（技术/文明水平）
- 只提取"对当前剧情有影响"的历史，不提取创世神话（除非角色会提及）
- 不提取与剧情无关的远古历史流水账

## 提取时的禁律操作

在从原文提取世界观信息时同步执行：

1. **去小说化** — "在这片广袤无垠的大地上，人族与妖族争斗了数千年" → "人族与妖族长期对立"
2. **去意象** — "中央神州是人类最后的堡垒，如同黑暗中的灯塔" → "中央神州：人族主战场"
3. **去主观** — "六大仙宗是修仙界最负盛名的门派" → "六大仙宗：剑宗、丹宗、玄天宗..."
4. **去连接词** — 所有"存在着""分别是""是一个""由...组成"用冒号/逗号替代
5. **去废话** — "充满机遇与挑战的世界""波澜壮阔的时代" → 删除
6. **去禁词** — 原文出现的"一丝""一缕""不易察觉""鲜明对比"等→不提取

---

## 提取输出格式

对每个提取维度，输出以下结构：

```
### [维度名称]
- 手法总结：<源材料在构建此维度时的写作手法>
- 关键原文：<1-3段原文例句，保持原句原貌，标注章节>
- 禁词标记：<原文中的意象/禁词/主观评价，标注改写方案>
- 条目化输出：
  <直接可用于世界书条目的压缩内容，用YAML或列表格式>
```

## 条目化示例

**原文：**
> "在这片名为'阴阳大陆'的土地上，人族与妖族争斗了数千年。中央神州是人类的最后堡垒，六大仙宗在此地设立道场。东荒则是妖族的领地，万妖国统治着那里的所有生灵。北原的魔土被称为天魔帝都，是魔族的发源地。"

**禁词标记：**
- "争斗了数千年" → AI已知修仙世界观有人妖之争，不提取
- "最后的堡垒" → 主观形容词，删
- "统治着那里的所有生灵" → 主观形容词，删

**条目化输出：**
```yaml
世界: 阴阳大陆，修仙世界观
区域:
  中央神州: 人族主战场，六大仙宗所在地
  东荒妖域: 妖族领地，万妖国统治
  北原魔土: 魔族发源地，天魔帝都
  西漠佛国: 佛门净土
  南疆巫地: 巫族势力
  四海: 龙族领地
```

西漠/南疆/四海是**补全**的——如果原文有提到相关信息。

## 压缩检查清单

世界观条目写完后逐条检查：
- [ ] 每行是一个信息点，不是叙述句
- [ ] 没有"存在着""分别是""是一个""由...组成"等连接词
- [ ] 没有主观评价（"强大的""神秘的""令人畏惧的"）
- [ ] 没有比喻和意象
- [ ] AI已知的信息已被删除
- [ ] 列表格式优先于段落格式
- [ ] 禁词已全部剔除（一丝/一缕/一抹/不易察觉/鲜明对比等）
- [ ] 另一个不了解这个世界的AI，仅凭此条目能还原设定
