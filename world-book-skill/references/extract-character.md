# 角色提取指南

从轻小说/游戏/小说等源材料中提取角色信息，转化为世界书条目（**XML 包裹 YAML** 格式）。

---

## 提取原则

**角色条目的内容来自原文，但格式必须按 `character-guide.md` 的 YAML 结构重写。** 提取是从原文找信息，写入是按规范写条目。

**零度写作 + 行为具体化贯穿始终。** 以下禁律在提取时—并执行：

---

## 提取禁律（在提取原文信息时同步执行）

1. **剔除意象比喻** — "眸如星辰" → "天青色瞳"；"嗓音如银铃" → "嗓音清脆"
2. **剔除解释性修饰** — "这个动作体现了他的紧张" → 不提取，用动作本身替代
3. **剔除不存在事物的描写** — "拂去不存在的灰尘" → 不提取
4. **剔除主观形容词** — "温柔""善良""强大"后必须找行为依据，无依据则不写入
5. **剔除禁词** — 规则同 `character-guide.md` 写作禁律
6. **剔除AI已知信息** — "黑发""白皙""年轻"等默认特征不提取

---

## 提取维度

### 1. 基本信息 + 外貌

从原文提取：

- **姓名/年龄/性别/身份**：直接提取，原文没写不补
- **外貌特征**：只提取**偏离AI数据库默认值**的特征
  - 判断标准：遮住名字，仅凭外貌描写能认出这个角色吗？
  - 提取什么：异色瞳、疤痕、纹身、特殊发色、标志性服装、体型显著特征、习惯饰品、声线特色
  - 不提取什么：默认外貌（"黑发""白皙"）、万能美人描写（"精致""好看"）、比喻意象（"眸如星辰"）
  - 禁提取：意象比喻必须转化为特征白描
- **声线/说话风格**：从原文对话中提取语气词、说话节奏、情绪变化时的声音变化

### 2. 性格

- **核心驱动力**：她最想要什么？最怕什么？（需要从角色行为反推）
- **性格特征**：每条都必须有**原文依据**——她做了什么/说了什么说明她是这样的人
- **喜欢/讨厌**：从角色行为和对话中提取
- **习惯性动作**：原文中反复出现的动作细节（转笔、咬唇、摸头发等）
- **隐藏面**：表里矛盾——原文中一个人时的表现、不让别人看到的一面

### 3. 背景

- 家庭背景（只提取有影响的部分）
- 关键经历（改变角色的转折事件）
- 不提取：流水账、无关童年琐事、不改变角色的日常

### 4. 能力

- 每个能力的**获得方式**（原文中的来源/触发事件）
- 每个能力的**具体效果**（原文中实际做了什么，转化为效果列表）

### 5. 关系

- 每个重要关系的**认识方式**
- **互动方式**——原文中两人实际做了什么
- **关系动态变化**——原文中关系如何演变

---

## 提取输出格式

```
角色：<角色名>

### 基本信息
- 原文依据：<标注章节>

### 外貌
- 原文依据：<原文描写引用，标注章节>
- 禁词标记：<标注原文中的意象比喻/禁词，说明如何转化>
- 提取：<去掉比喻/默认值/禁词后，只留特征>

### 性格
- 原文依据：<角色行为/对话引用>
- 核心驱动力：<从行为反推>

### 背景
- 原文依据：<关键事件引用>
- 提取：<只留改变角色的关键事件>

### 能力
- 原文依据：<能力描写引用>
- 效果列表：<转化为效果描述>

### 关系
- 原文依据：<互动描写引用>
```

## 提取示例

**原文（来自小说多处，汇总）：**
> "仁科鸟子，20岁，金发蓝眼，身高一米六八。她在温哥华出生，父母都是加拿大军人，已去世。父母死后移居日本，被家教老师闰间冴月带入了一个叫'里世界'的异空间。冴月大约九个月前在里世界失踪了，此后鸟子一个人反复进入里世界找她。在草海里她救下了濒死的空鱼——这是她在里世界遇到的第一个活着的人。她的左手是透明的——在一次与里世界生物的接触中被感染了，能触碰和破坏里世界的东西。鸟子性格开朗热情但极度害怕一个人待着——'重要的人总是突然消失'。她对空鱼说过'我最喜欢你了'，说的时候口气不容置疑。"

## 提取注意事项

### 原文没有的字段
- **不补**。`three_sizes` 原文没写就不写，`skin` 没有特殊皮肤特征就不写，`mbti` 不确定就不写
- 标注 `<!-- 原文未提及 -->` 在 outline.txt 中

### 原文写了但是冗余的
| 原文 | 提取 |
|------|------|
| "精致的脸蛋，白皙的皮肤" | 不提取（AI已知18岁女生皮肤好） |
| "眸如星辰般灿烂" | 提取为"天青色瞳"或"眼睛明亮"，去掉比喻 |
| "发梢在强光下泛起浅金光点" | 提取为"金发"，去掉美文 |
| "她是个热心肠的人" | 转化为行为依据：她做了什么让人觉得热心 |
| "一丝温柔的微笑""弯起嘴角" | 不提取此描写，提取为具体的行为互动 |

### 与原文的"信任距离"

原文说"她很温柔"→ 找原文中她"温柔"的具体表现，用行为写性格。如果原文只有形容词没有行为，则不写该性格。
