# 故事/章节提取指南

从轻小说/游戏/小说等源材料中提取章节故事，转化为世界书条目。

---

## 核心原则

- **一章一个条目**，不合并、不省略
- **故事不需要总纲条目**——每个章节条目独立存在
- 所有章节条目放在世界书末尾（order=200+）
- 章节条目用叙事性结构，保留原文情节原貌，但精炼压缩
- **提取时剔除禁词和意象比喻**——章节条目也应遵循零度写作

---

## 条目结构

每个章节条目采用 **XML 包裹 YAML** 格式：

```yaml
<story_chapter>
title: File N：章节标题——标志性事件
timeline: 时间标注（朝代/季节/年份/相对时间线）
location: 主要发生地点
full_scene:
  - 段落标题: 该段的情节叙述。用精炼语言交代发生了什么，保留关键动作和对话要点。不写心理活动，只写事件本身。
  - 段落标题: 同上。每个段落对应一章内的一个场景或事件段落。按剧情推进顺序排列。
available_characters: 本章出现的、可用于RP的角色/NPC/生物名，逗号分隔
key_revelations: 本章的核心剧情揭示/转折点，逗号分隔
rp_hooks:
  - 可以作为新增角色插入本章剧情的方式
  - 另一种切入角度
</story_chapter>
```

### 字段说明

| 字段 | 写法 | 说明 |
|------|------|------|
| `title` | `File N：标题——事件概述` | 章节编号 + 标题 + 标志性事件作为副标题 |
| `timeline` | 简短时间标注 | 季节/年份/时间线节点，如"次年冬""第三章后两周" |
| `location` | 具体地点 | 本章主要发生地，如"新宿歌舞伎町·巴厘岛主题情侣酒店" |
| `full_scene` | `- 段落标题: 叙述内容` | 每个项是一个场景段落，只写发生了什么 |
| `available_characters` | 逗号分隔 | 本章出现的可用于RP的角色（不含已有详细条目主角） |
| `key_revelations` | 逗号分隔 | 本章的核心剧情揭示——改变读者认知的信息 |
| `rp_hooks` | `- ` 列表 | 供RP插入的切入点，1-3条 |

---

## full_scene 写法规则

- 格式：`- 段落标题: 叙述内容`
- 段落标题来自该段的标志性事件或关键词
- 每段 50-150 字，精炼叙事，只写"发生了什么"
- **不写角色内心独白**，不写环境描写的美文，只写事件和动作
- 关键对话保留要点或标志性台词（一句即可）
- 场景段落按剧情推进顺序排列
- **禁词和意象比喻在压缩时剔除**——不保留原文中的比喻修饰

## 提取示例

**原文（缩减版）：**
> "次年冬天，空鱼和汀正式签约成为牧场的共同管理人。汀设定了一个奇怪的经费制度——没有上限，但需要两个人共同同意。签完约，空鱼为了避免和鸟子独处，叫上了小樱、茜理和夏妃一起去了新宿的情侣酒店。五个人喝酒吃蜂蜜土司，空鱼喝多了开始抱怨酒店太贵。这时茜理在洗澡时从门口瞥见了一个极高的黑发白衣女鬼——Pontianak。与此同时夏妃在桑拿房被拍了四次肩膀。空鱼判断还没说过话，所以暂时安全。但她喝醉后右眼的力量失控了。巴厘岛风格的巴隆舞狮从门口闯入，配合着甘美朗音乐。空鱼脱光后以失神状态跳起了'魔女让特之舞'——在场所有人全部陷入集体恍惚，脱光跳舞。早上醒来一片混乱，五人赤身裸体躺在各处。她们统一口径当作什么都没发生。事后鸟子对空鱼说：'我感觉就只有我在那瞎起劲似的，你不愿意单独跟我去，所以把别人都叫上了。'空鱼这才意识到——鸟子想要的女子会是只有两个人。"

**条目化输出：**
```yaml
<story_chapter>
title: File 16：情侣酒店女子会——Pontianak与巴隆舞狮集体失神事件
timeline: 次年冬
location: 新宿歌舞伎町·巴厘岛主题情侣酒店
full_scene:
  - 管理合同: 空鱼与汀正式签约成为牧场管理人。汀设模糊经费制度——不设上限但需双方同意。汀对空鱼的评价："在纸越小姐看上去很顺从的时候，必须思考她是否在看不到的地方做着可怕的事。"
  - 女子会: 空鱼为避免与鸟子独处，叫上小樱、茜理、夏妃共五人在新宿歌舞伎町的巴厘岛主题情侣酒店举行女子会。五人喝酒、吃蜂蜜土司、洗澡。空鱼喝醉后吐槽"为什么情侣酒店这么贵——"
  - Pontianak事件: 茜理在洗澡时从门口瞥见Pontianak——个子高到快顶到门框、黑色长发、苍白面孔、穿白色衣服的印尼女幽灵。夏妃在桑拿房被拍肩膀四次——四与"死"同音。空鱼判断只要没说过话就没问题。
  - 巴隆舞狮集体失神: 空鱼喝醉后右眼力量失控。巴厘岛风巴隆舞狮闯入——四条腿大影子、通红脸、圆圆巨眼、金冠。配合甘美朗音乐和卡恰舞节奏。空鱼脱光后以失神状态跳起魔女让特之舞。在场所有人陷入集体恍惚全部脱光跳舞。早上醒来一片混乱，五人赤身裸体躺在各处，统一口径当作什么都没发生。
  - 鸟子的伤心: 鸟子对空鱼的反应感到极度伤心："我感觉就只有我在那瞎起劲似的。你不愿意单独跟我去，所以把别人都叫上了。"空鱼才意识到鸟子想要的女子会是"只有两个人"。
available_characters: Pontianak、巴隆舞狮
key_revelations: 空鱼右眼酒精失控、巴隆舞引发集体失神、鸟子感情受伤
rp_hooks:
  - 作为女子会的第六名成员，同样被巴隆舞狮影响陷入集体失神
  - 事后帮助五人统一口径，处理酒店赔偿和安保问题
</story_chapter>
```

## 世界书配置

| 设置 | 值 |
|------|-----|
| position | 1 (↓Char) |
| 激活方式 | 绿灯（关键词触发） |
| order | 200 + 章节号（区分于角色/物品的50-100） |
| 递归 | `--prevent-recursion --exclude-recursion` |
| 扫描深度 | `--scan-depth 2` |

### 关键词设计

每章条目的关键词覆盖本章的核心触发词：

```
--keys "第16章,File16,情侣酒店,女子会,Pontianak,巴隆舞狮"
```

覆盖原则：
- 章节号（"第16章""File16"）——用户可能在对话中引用章节
- 标志性事件名（"女子会""集体失神"）
- 标志性地点（"情侣酒店"）
- 本章独有的角色/NPC/怪物名（"Pontianak""巴隆舞狮"）

## 提取步骤

### 1. 通读章节，标注段落分界

用段落标题为本章划分场景段落，标准：地点切换 / 主要人物切换 / 事件段落结束。

### 2. 每个段落提炼为精炼叙述

- 去掉环境描写的美文
- 去掉角色内心独白
- 去掉意象比喻和禁词
- 保留标志性动作和对话要点（一句）
- 每段 50-150 字

### 3. 提取元信息

- `available_characters`：本章出场的、不是主角的、可用于RP的角色/NPC
- `key_revelations`：本章揭示的新信息或转折（改变读者/角色认知的信息点）
- `rp_hooks`：思考"如果{{user}}要插入这一章的剧情，有哪些切入角度"

### 4. 写入条目 + 设置关键词

## 与其他条目的关联

故事章节条目中的角色/NPC/怪物，如果世界书中有详细信息条目，可用 `@名称` 引用：

```yaml
full_scene:
  - Pontianak事件: 茜理瞥见 @Pontianak——个子高到快顶到门框、黑色长发、苍白面孔
```

使用 `python scripts/query.py <世界书路径> --resolve` 验证引用。

## 检查清单

- [ ] 每章一个条目，无遗漏
- [ ] 段落标题分段清晰，按剧情顺序排列
- [ ] full_scene 中只写事件，不写美文和心理描写
- [ ] full_scene 中无禁词和意象比喻
- [ ] title 包含标志性事件副标题
- [ ] available_characters 列出的角色在世界书中有对应条目
- [ ] key_revelations 覆盖了本章所有重要揭示
- [ ] rp_hooks 提供 1-3 个可操作的RP切入点
- [ ] 配置：position=1, 绿灯, order=200+, scan-depth=2
- [ ] 双递归：`--prevent-recursion --exclude-recursion`
- [ ] 关键词覆盖章节号、标志性地点/人物/事件
